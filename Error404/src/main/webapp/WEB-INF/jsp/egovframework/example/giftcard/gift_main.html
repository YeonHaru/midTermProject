기프트카드 이용안내

① 등록하기

문자(MMS)로 카드번호를 받았거나,
매장에서 기프트카드를 구매했다면?

카카오톡 선물하기, 기프티콘 등
기프트카드 교환번호를 받았다면?

② 충전하기

등록된 기프트카드의 잔액을 추가 충전하거나,
자동충전을 신청하고 싶다면?

③ 온라인에서 이용하기

주문/결제 단계 내 결제수단에서
기프트카드 선택 → 등록된 카드 중 이용할 기프트 카드 선택하여 결제

④ 매장에서 이용하기

기프트카드는 전국 교보문고 매장에서
상품 구매 시 사용 가능합니다.

  • 문자(MMS)로 전달받은 이미지 제시
  • 온라인 등록 완료 후 기프트 카드의 바코드를 제시
